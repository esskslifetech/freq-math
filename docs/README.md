## Project Overview

Freq-Math is a unique application that transforms mathematical expressions into rich audio experiences. It combines:

- **High-performance C++ backend** for equation parsing and evaluation
- **Flexible Python frontend** for audio synthesis and visualization
- **Secure expression processing** with AST-based validation
- **Multiple synthesis modes** for diverse sound creation
- **Real-time visualization** of mathematical functions and audio analysis

### Key Features

- 🔢 **Real-time Equation Evaluation**: Parse and evaluate complex mathematical expressions instantly
- 🎵 **Mathematical Sonification**: Convert equations into audio using multiple synthesis modes
- 📊 **Interactive Visualization**: Real-time plots of functions, waveforms, and spectra
- 🔒 **Secure Processing**: AST-based security validation prevents code injection
- ⚡ **High Performance**: C++ backend with parallel processing
- 🎛️ **Multiple Synthesis Modes**: FM, AM, phase distortion, wavetable, and direct mapping
- 📝 **Rich Equation Support**: Trigonometric, exponential, logarithmic, and custom functions
